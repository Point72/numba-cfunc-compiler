## Pre-requisites

You need Python >=3.10 on your machine to install `numba-cfunc-compiler`.

## Install with `pip`

```bash
pip install numba-cfunc-compiler
```

## Install with `conda`

```bash
conda install numba-cfunc-compiler --channel conda-forge
```

## Source installation

For other platforms and for development installations, [build `numba-cfunc-compiler` from source](Build-from-Source).
